# UX Designer Skill

## Goal
Design calm, minimal, responsive UX for the meditation app.

## Product principles
- Serious, peaceful, uncluttered
- Reduce taps
- Avoid feature bloat
- Respect meditation context
- Make common actions obvious
- Work well on mobile, tablet, and desktop

## Always consider
- entry point
- primary task
- decision points
- validations
- edge cases
- empty states
- error states
- success states
- responsive behavior by breakpoint
