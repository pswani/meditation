# 99-merge-branch.md

## Goal
Safely merge the iPhone Safari real-device QA branch after review and verification.

## Preconditions
- No unresolved blocker/high/medium findings.
- Required docs and session handoff updates are complete.
- Any code changes have passed the required automated checks.

## Steps
1. Confirm branch is `chore/ios-safari-real-device-qa`.
2. Sync with parent branch per team policy.
3. Resolve conflicts carefully.
4. Merge back to parent branch.
5. Update `requirements/session-handoff.md` with the latest QA state.

## Final output
- Merge summary
- commits included
- QA artifact paths
- remaining known gaps
