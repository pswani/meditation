# 00-create-branch.md

## Goal
Create an isolated branch for codifying and, when possible, executing real-device iPhone Safari QA.

## Required reading
Review before changes:
- README.md
- AGENTS.md
- PLANS.md
- docs/product-requirements.md
- docs/ux-spec.md
- requirements/decisions.md
- requirements/session-handoff.md
- docs/review-ios-safari-ux-issues.md
- docs/review-ios-safari-ux-hardening.md
- docs/test-ios-safari-ux-hardening.md

## Branch setup
1. Confirm clean working tree.
2. Create and check out:
   - `chore/ios-safari-real-device-qa`
3. Record parent branch and current commit.

## Output
- Branch created.
- One short summary of the manual validation gaps this bundle will close.
