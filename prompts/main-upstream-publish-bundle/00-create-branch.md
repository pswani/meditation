# 00-create-branch.md

## Goal
Create an isolated branch for preparing the current local `main` state for upstream publication.

## Required reading
Review before changes:
- README.md
- AGENTS.md
- PLANS.md
- requirements/decisions.md
- requirements/session-handoff.md

## Branch setup
1. Confirm clean working tree.
2. Create and check out:
   - `chore/main-upstream-publish`
3. Record parent branch, current commit, and `main` vs `origin/main` ahead/behind state.

## Output
- Branch created.
- One short summary of what is pending before `main` can be pushed upstream.
