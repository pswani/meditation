# 00 Create Branch

Create a new branch for full cross-platform `sankalpa` parity.

Requirements:
- Branch name should start with `codex/`.
- Suggested branch: `codex/sankalpa-full-parity-feature`
- Confirm the working tree state first and avoid disturbing unrelated changes.
- Review the required docs before starting:
  - `README.md`
  - `requirements/intent.md`
  - `docs/product-requirements.md`
  - `docs/architecture.md`
  - `docs/ux-spec.md`
  - `docs/screen-inventory.md`
  - `requirements/roadmap.md`
  - `requirements/decisions.md`
  - `requirements/session-handoff.md`
  - `PLANS.md`

Follow-through:
- Create or refresh one active ExecPlan at `docs/execplan-sankalpa-full-parity-feature.md`.
- Then continue with `01-implement-sankalpa-full-parity.md`.
