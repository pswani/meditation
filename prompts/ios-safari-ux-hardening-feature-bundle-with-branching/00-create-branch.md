# 00-create-branch.md

## Goal
Create an isolated branch for iPhone Safari UX hardening work after the initial lock-screen bell mitigation.

## Required reading
Review before changes:
- README.md
- AGENTS.md
- PLANS.md
- docs/product-requirements.md
- docs/ux-spec.md
- requirements/decisions.md
- requirements/session-handoff.md
- docs/review-ios-safari-ux-issues.md

## Branch setup
1. Confirm clean working tree.
2. Create and check out:
   - `fix/ios-safari-ux-hardening`
3. Record parent branch and current commit.

## Output
- Branch created.
- One short summary of the pending issues this bundle will address.
